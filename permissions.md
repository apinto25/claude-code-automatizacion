# Permisos Claude Code

## Bash
Comandos de terminal

```
"Bash(pytest *)"
"Bash(git status)"
```

## Read / Edit / Write
Acceso a archivos

```
"Read(.env)"           → lectura de archivos
"Edit(src/**)"         → edición de archivos
"Write(src/**)"        → creación de archivos
```
